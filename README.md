# crewops
CrewOps - Production crew scheduling &amp; budget management platform. Real-time crew allocation, project budgeting, and invoice generation for film, TV, and event production companies.


## Getting Started

Visit the live app at: https://crewops-iota.vercel.app

### Features
- Dashboard with real-time crew & project overview
- - Crew member management with roles and hourly rates
  - - Project creation and budget tracking
    - - Real-time labor cost calculations
      - - Invoice generation and tracking
        - - Data persistence with localStorage
